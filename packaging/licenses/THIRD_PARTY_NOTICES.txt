ClearPodcast license notice sources
===================================

Platform release artifacts should copy a platform-specific generated notice
directory from this folder, not this compatibility note.

Current platform notice sources:

- `packaging/licenses/windows-x64/THIRD_PARTY_NOTICES.txt`

Regenerate the Windows notice file with:

```text
npm run licenses:windows-x64
```

The Windows staging script runs that command automatically before copying
license notices into the portable artifact.
